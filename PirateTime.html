Pirate Time - Art

Pirate Time was developed as a class project for UW-Whitewater starting in mid November of 2022. I worked on it with one other classmate, we separated the work with one person handling the coding of the project and one person worked on art. I took on the art aspect of this project. All of the art for this project is original and was done in the month period we had for the final. I was in charge of the art as well as the animations for all player characters and enemies. I used Unity's built in skeletal animation tool to create the animations for all the characters in the game. All the art of the game is original and sketched out in Photoshop.

Pirate Time - Programming

Along with all of the visual aspects of the project I also handled some of the coding elements as well. I created all of the game and scene managers that would use the singleton pattern to exist through different scenes including the Audio Manager, Player Manager, and Game Manager. I also created the menu scenes as well as the player select screen as shown above. The character select screen works by using the player manager to record which character the player selects then spawns the selected player in at the beginning of each playable scene.
